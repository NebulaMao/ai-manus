## 📊 项目核心信息

### 产品愿景
AI Manus 是一个通用的 AI Agent 系统，支持在沙箱环境中运行各种工具和操作，为用户提供安全、可控、高效的 AI 助手环境。

### 技术架构
- **前端**: Vue 3 + TypeScript + Tailwind CSS
- **后端**: FastAPI + Python + MongoDB + Redis
- **容器**: Docker + Docker Compose
- **通信**: SSE + WebSocket

### 开发计划
- **开发周期**: 12周 + 1周准备
- **团队规模**: 7人
- **发布目标**: 2026年2月
- **MVP目标**: 100 MAU，30日留存率≥20%

### 核心功能
- 🤖 AI Agent 管理与配置
- 🔒 安全沙箱执行环境
- 🖥️ 响应式Web用户界面
- 🛠️ 核心工具集成
- 📊 实时协作与监控
